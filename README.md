## Requirements

- GSUBZ API Key.
- Node JS v18 or higher.

## Installation

```sh
npm install @specialman/gsubz
```

## Setup

```javascript
let gsubz = require("@specialman/gsubz");
gsubz = gsubz("your_api_key");
```
